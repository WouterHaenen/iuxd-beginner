Exercises.

Practice makes perfect, or that’s what they say atleast. Quite frankly, we believe in it too! In order to get good at something you must apply yourself regularly. To facilitate this we’ve set up some exercises for you to get going! You’ll find exercises in varying difficulty, from beginner to expert. And everything in between. You might even make up your own and send it to us! It might be featured, no promises though.

Advanced

Instagram adjustment. In this exercise you will interact with different elements in succession. Make sure that by providing one input, multiple elements will start moving. Make sure you can reverse this movement. Start by making the first interaction and linking the movement of the image. If you can’t seem to figure it out make sure to check out the literature on the origami.design website.
